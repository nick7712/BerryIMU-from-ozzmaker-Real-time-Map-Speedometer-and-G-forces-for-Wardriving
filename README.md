# BerryIMU-from-ozzmaker-Real-time-Map-Speedometer-and-G-forces-for-Wardriving
wrote a map that hosts on a local webpage, and you can start the script which will collect public wifi data
"Raspberry Pi Automotive Telemetry & Wardriving Platform"A custom Python platform built for the Raspberry Pi 4 utilizing an OzzMaker BerryIMU and a GNSS GPS module. Features a live local web dashboard (Plotly/Dash) displaying a digital speedometer, a longitudinal G-Force meter, and 3D tilt tracking alongside an independent background wireless router network logging engine.
